# GenAI-Cricket-Commentary

> AI-powered cricket commentary generation leveraging computer vision, large language models, and text-to-speech.

## Overview

GenAI Cricket Commentary is an end-to-end AI system that automatically generates cricket commentary from match videos. The pipeline combines computer vision for video understanding, large language models for context-aware commentary generation, and text-to-speech to produce synchronized spoken commentary. The project demonstrates the integration of multimodal AI techniques to automate sports commentary generation.
